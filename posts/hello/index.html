Welcome!

I’m writing here because I’ve decided to return to coding. Coding has missed me (LOL).

I started messing around with code in 1982, using a Sinclair ZX81. After uni, I worked as a software / web developer for six years. So I’ve been around for quite some time. After a while I reached a phase in my life when I never wanted to look at a screen ever again. Splitting headaches, RSI, the works. That was in 2003. Things have changed since then. Screens are easier to look at these days (no more CRT tubes flickering away, a lot less eye strain). Keyboards are easier on the hands. And I must admit, I enjoy the challenges of dealing with code and logic much more than the day job. So I’ve decided to make a comeback, and here’s where it starts.

I finally have a week’s free time (unpaid), so I’m going to learn lots of stuff. First will be to make a static website using Hugo.

I guess I’ll have to learn some markdown before I can really get going with this Hugo site. There’s nothing too difficult or scary about that though. I’m sure it will all work out fine.
